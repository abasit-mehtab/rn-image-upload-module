# rn-image-upload-module

Simple React Native module to upload an image data to a server.

## Installation

```sh
npm install rn-image-upload-module
```

## Usage

```js
import { multiply } from 'rn-image-upload-module';

// ...

const result = await multiply(3, 7);
```
